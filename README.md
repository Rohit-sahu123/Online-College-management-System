# Online College Management System

This is my college project. It is a web-based College Management System made using **Java, JSP, Servlet and MySQL**.

## Features
- Student management
- Teacher management
- Course and subjects
- Login system
- Result and records

## Tech Stack
- Java
- JSP / Servlet
- HTML, CSS, JS
- MySQL
- Apache Tomcat
